# Esempi Multilingue - TermInA AI

Questi esempi dimostrano come l'AI di TermInA risponde automaticamente nella stessa lingua in cui viene posta la domanda.

## 🇮🇹 Italiano

### Domanda
```
ai: Come posso creare una nuova cartella sul desktop?
```

### Risposta Attesa
```
Ti aiuto a creare una nuova cartella sul desktop. Posso eseguire il comando per te:

mkdir ~/Desktop/nuova_cartella

Questo comando creerà una cartella chiamata "nuova_cartella" sul tuo desktop. Vuoi che lo esegua automaticamente?
```

## 🌐 Testo Misto

Il sistema gestisce anche testo che mescola più lingue, scegliendo la lingua dominante:

```
ai: Hello, vorrei creare a new folder. Grazie!
```

In questo caso, il sistema rileverà l'italiano come lingua dominante e risponderà in italiano.

## 💡 Suggerimenti per l'Uso

1. **Usa una sola lingua per domanda** per risultati più precisi
2. **Frasi complete** migliorano il rilevamento della lingua
3. **Termini tecnici** sono riconosciuti in tutte le lingue supportate
4. **Il sistema mantiene la lingua** durante tutta la conversazione
5. **Fallback automatico** all'inglese se la lingua non è riconosciuta
